What is a Program?

A program is a set of instructions that you write to tell a computer what to do

What is programming?

Programming is the task of writing those instructions in a language that the computer can understand.

How Computers ‘Think’

The short answer is that they don’t think.

The slightly longer answer is that while computers don’t think, they act as if they do, by sequentially executing simple instructions.

The only things a computer knows are the things we tell it.

Pseudo code

Pseudocode is the process of writing a program without using the syntax of a programming language.

Note: We will talk about algorithms later.

Pseudocode is a mixture of natural language and high-level programming constructs. For example, If the door is closed and I want to exit the room, then open the door. This class we will learn what these keywords and structures are and how they are used to create a program…

Pseudo code example


//  HOW TO CREATE USER'S FULL NAME
//  Get the user's first name
//  Get the user's last name
//  Add the first name
//  And a space
//  And then the last name

Arrays

What is an array?

Why would we use it?

Arrays

Declaring a variable


var exampleVariable = "Cat";

Declaring an empty array


var exampleArray = [];

Declaring an array with items in it


var exampleArray = ["Cat", 34, "This sentence", true];

Arrays

Arrays can contain other arrays


var exampleVariable = ["Cat", 34, "This sentence", ["Inner Array Item", 45]];

They can also contain variables


var innerArray = ["Inner Array Item", 45];
var exampleVariable = ["Cat", 34, "This sentence", innerArray];

Arrays

Let's create an array

Lets create a car in javascript. We could use a variable...


var car = "Jeep Cherokee";

Well a car has more than a model so lets add a year and color. We could use more variables...


var carModel = "Jeep Cherokee";
var carYear = 2001;
var carColor = "Blue";

But our data is all individual variables and not grouped to make a person car. We could use an array...


var car = ["Jeep Cherokee", 2001, "Blue"];

Arrays

How do we access the individual elements?


var car = ["Jeep Cherokee", 2001, "Blue"];

Each position in the array has an index.

The index starts at 0 then 1 then 2 then 3...


var car = ["Jeep Cherokee", 2001, "Blue"];

car[0] // "Jeep Cherokee"
car[1] // 2001
car[2] // "Blue"

We can add to an array this way too


var car = ["Jeep Cherokee", 2001, "Blue"];

// add whether the car is active or not (true or false)
car[3] = false; 

car // ["Jeep Cherokee", 2001, "Blue", false]

Lets create a person in javascript. We could use a variable...

But an object could be even better...


var person1 = {
     species: "human",

Objects

How do we access the individual elements?


var car = { brand: "Toyota", year: 1999, topSpeed: 140, ecoFriendly: false };

Each property is the index.

Remember how we accessed individual elements of an array?


var person = ["John Doe", 50, "Blue"];
person[0] // "John Doe"

For objects we can do...


var car = { brand: "Toyota", year: 1999, topSpeed: 140, ecoFriendly: false };
car["brand"] // "Toyota"
car["year"] // 1999

For objects we can ALSO do...


var car = { brand: "Toyota", year: 1999, topSpeed: 140, ecoFriendly: false };
car["brand"] // "Toyota"
car.brand // "Toyota"

car["year"] // 1999
car.year // 1999

Objects

Lets look at how to access deeply nested data


var student = { 
  name: "Michael Finneran"; 
  classes: ["HTML", "CSS", "JS"]; 
  car: { 
    brand: "Toyota", 
    year: 1999, 
    topSpeed: 140, 
    ecoFriendly: false
  }
};

student.name // "Michael Finneran"
student.classes // ["HTML", "CSS", "JS"]
student.classes[0] // "HTML"
student.classes[1] // "CSS"
student.classes[2] // "JS"
student.car // { brand: "Toyota", year: 1999, topSpeed: 140, ecoFriendly: false}
student.car.brand // "Toyota"
student.car.year // 1999
