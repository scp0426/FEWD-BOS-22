For Loop

For

    code block to be executed
  }

  }

What is jQuery?

Created by John Resig in 2005, released in January of 2006.

Built in an attempt to simplify the existing DOM APIs and abstract away cross-browser issues.

jQuery is simple (and easy!)


$('div').addClass('special');

Find some element on the page (DOM tree)

Do something with it.

Let's break this down...


$('div').addClass('special');

$ is the jQuery Object (also named jQuery)

"div" finds some elements using CSS selectors

$('div') returns a jQuery set (containing 0 to many DOM nodes).

in javascript we had to do document.getElementsByTagName('div')

addClass(...) modifies all the divs

Wait, what if that selector doesn't exist?


$('.random-div').addClass('special');

Just like CSS, if jQuery can't find that element, nothing happens

jQuery finds elements in the DOM tree on the page to manipulate, so the "tree" needs to be there before it can do it's magic right?

jQuery "ready" function


$(document).ready(function(){ 
    // Your jQuery code goes in here 
});

jQuery has a ready event that fires the instant the DOM is ready to be worked with.

Stick all your jQuery code inside the "ready" block so that it fires when it should
