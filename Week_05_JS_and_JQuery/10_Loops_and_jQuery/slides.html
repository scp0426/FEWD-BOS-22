Variables

What are variables?

Variables

  • We can tell our program to remember values for us to use later on.
  • The action of saving a value to memory is called assignment
  • The entity we use to store the value is called a variable
  • The action of getting the value from a variable is called accessing the variable

Variables Declation

Declaration


var age;

Assignment


age = 21;

Both


var age = 21;

Variables Re-assignment


var name = "Laura";
          

name = "Mike";

Note: Now the variable "name" equals "Mike".

Variable Conventions

Variables start with a lower case letter.

If they contain multiple words, subsequent words start with an upper case letter.


var numberOfStudents = 10;

Variables & Data Types

What can you store in variables?

Supported Data Types

  • String = text
  • Int / Float = number
  • Boolean = true/false

Loops

While Loop


while (condition) {
    code block to be executed
}

While Loop


while (i < 10) {
    text += "The number is " + i;
    i++;
}

For Loop

For


  for (counter, limit; condition; counter_increment) {
    code block to be executed
  }

Example



  for (var counter = 0, limit = 5; counter < limit; counter++) {
      console.log("The number is " + counter);
  }

Example


  var animalArray = ["Cat", "Dog", "Fish"];

  for (var counter = 0, limit = animalArray.length; counter < limit; counter++) {
      console.log("The animal is " + animalArray[counter)];
  }

Example


  var animalArray = ["Cat", "Dog", "Fish"];

  for (var i = 0, j = animalArray.length; i < j; i++) {
      console.log("The animal is " + animalArray[i)];
  }
