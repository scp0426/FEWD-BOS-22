How do we learn?

  1. Use the documenation
  2. Find blog tutorials/introductions
  3. Find video tutorials/introductions
  4. Ask StackOverflow and Google
  5. Build an example first before applying to a project

Bootstrap

  1. Use the documenation: getbootstrap.com
  2. Find blog tutorials/introductions http://www.w3schools.com/bootstrap/ and others
  3. Find video tutorials/introductions https://www.youtube.com/playlist?list=PL4cUxeGkcC9g_69kOfXICzT_hZ79_td99 and others
